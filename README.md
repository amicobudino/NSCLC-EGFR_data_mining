# NSCLC-EGFR_data_mining

The aim of the project is to analyze from a genomic and transcriptomic point of view the Ecauriotic Growth Factor Receptor (EGFR) gene in Non Small Cell Lung Cancer (NSCLC).

In particular the point of invesitgation is the down modulation of EGFR in osimertinib resistant lung adenocarcinoma. 

Read the detailed report at the `report_NSCLC_EGFR.pdf`file.
